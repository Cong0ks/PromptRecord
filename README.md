# 提示词记录器 Chrome 插件

一个简单易用的 Chrome 浏览器插件，用于记录和管理 AI 提示词。点击插件图标后，侧边栏会从浏览器右侧展开，方便快捷地管理提示词。

## 功能特性

- ✅ **侧边栏界面**：从浏览器右侧展开，不遮挡页面内容
- ✅ **提示词管理**：创建、编辑、删除提示词
- ✅ **自定义分类**：自由添加和管理提示词分类
- ✅ **组标签**：用逗号分隔的标签系统，可给提示词打多个标签
- ✅ **日期追踪**：自动记录创建和更新时间，智能显示相对时间
- ✅ **搜索过滤**：支持按标题/内容搜索，按分类和组筛选
- ✅ **本地存储**：使用 Chrome Storage API，数据安全存储在本地

## 安装方法

### 1. 准备图标文件

在 `icons/` 目录下放置以下三个尺寸的图标文件（PNG格式）：
- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

> 提示：可以使用在线工具如 https://www.favicon-generator.org/ 生成不同尺寸的图标

### 2. 加载插件到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹
6. 插件安装完成！

## 使用说明

### 创建提示词

1. 点击浏览器工具栏中的插件图标
2. **侧边栏会从浏览器右侧展开**
3. 点击"+ 新建"按钮
4. 填写以下信息：
   - **标题**：提示词的简短描述
   - **内容**：完整的提示词文本
   - **分类**：选择或输入分类名称（如：写作、编程、翻译等）
   - **组标签**：用逗号分隔的标签（如：工作, AI, 翻译）

### 编辑和删除

- 点击任意提示词卡片打开编辑窗口
- 可以修改所有字段后保存
- 点击"删除"按钮可删除该提示词

### 搜索和筛选

- **搜索框**：实时搜索标题和内容
- **分类筛选**：按分类过滤提示词
- **组筛选**：按组标签过滤提示词

## 文件结构

```
├── background.js      # 后台脚本（处理侧边栏打开）
├── sidepanel.html     # 侧边栏 HTML 页面
├── sidepanel.css      # 侧边栏样式文件
├── sidepanel.js       # 侧边栏主要逻辑代码
├── icons/             # 图标文件夹
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
├── CLAUDE.md          # 项目说明
└── README.md          # 说明文档
```

> **注意**：旧版本的 `popup.html/css/js` 文件已保留，如不再使用可删除。

## 数据结构

每个提示词包含以下字段：

```javascript
{
  id: string,           // 唯一标识符
  title: string,        // 标题
  content: string,      // 提示词内容
  category: string,     // 分类
  groups: string[],     // 组标签数组
  createdAt: string,    // 创建时间 (ISO 8601)
  updatedAt: string     // 更新时间 (ISO 8601)
}
```
